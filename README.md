  <h3 align="center">Workopia</h3>

   <div align="center">
     Workopia is a job listing application built with PHP. It includes a custom Laravel-like router, controller classes, views, a database layer and a project structure using namespaces and PSR-4 autoloading.

## <a name="tech-stack">⚙️ Tech Stack</a>

- PHP 7.4 or Higher
- MySQL 5.7 or Higher
- TailwindCSS

## <a name="features">🔋 Features</a>

👉 **User Authentication:** Custom built user authentication, login and register functionality

👉 **Listings (CRUD):** Comprehensive functionality for creating, reading, updating, and deleting listings, giving users full control over listing management.

👉 **Search & Filter:** Empowering users with a robust search and filter system, enabling them to easily find the job listings that match their preferences.

and many more, including code architecture and reusability
